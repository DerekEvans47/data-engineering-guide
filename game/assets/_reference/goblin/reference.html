PRODUCTION SPEC (matches the game engine)

• Frame height: 110 px baseline. Taller sheets are allowed when a pose needs headroom (goblin attack = 118 px) — pair with a per-anim scale override in TD_ENEMY_SHEET_IMAGES so pixels-per-source-pixel stays constant.
• Horizontal strip, single row, uniform frame width.
• Frame budgets: walk 2 (A-B loop) · death 4 (one-shot) · attack 4 (windup / strike / follow-through / recover).
• Transparent RGBA. No baked ground shadow (engine draws its own).
• Feet on a consistent baseline across ALL sheets — the engine hot-swaps animation states mid-frame.
• Side view facing right; engine mirrors for left.
• Style: hand-painted low-res sprite — soft blended shading and anti-aliased edges are CORRECT (this is not strict indexed pixel art).

DO NOT

• Brighten / saturate the skin toward lime or cartoon green.
• Thin out the dark halo outline — it is the style's signature.
• Change proportions (oversized head, stubby limbs stay).
• Add gear, weapons, colors not present in the frames above.
• Bake in shadows, glows, motion blur, particles, or checkerboard "transparency" — use a flat magenta background if true alpha is unavailable.
• Put any text or labels inside generated sheets.
