a session URL to the PR body, regardless of what was passed in the `body`
parameter — this is **not** something the calling prompt controls; it's
injected server-side by the GitHub MCP tool itself. CLAUDE.md's absolute rule
bans this exact pattern (session URLs + "_Generated by Claude Code_" footers).

strip the footer before merging.** Next session: budget for this extra step
